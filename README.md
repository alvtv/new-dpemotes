# dpEmotes

- This is a retake from the original dpEmotes, but this time it includes a protected event, so that those pesky modders cannot make your entire server dance.

# Installation

* To install this resource, follow these steps:

- Clone this repository
- Add the folder to your resources folder
- Run the SQL, or disable SqlKeybinding in Config.lua
- Add `start dpEmotes` to your server.cfg

And you're done!
